An admin dashboard with medieval aesthetics to sort, search, filter and expell Hogwarts' students.

This is a thorough KEA project of the third semester, to improve our vanilla JS abilities and expand our knowledge of sorting, mapping and filtering. The main idea behind this was to create a dashboard so Hogwart's administrator could have an overlook of the students.

Visit site Hacking Hogwarts

Organizing the sections

I divided the dashboard in 3: the top part is a scrollable bar that allows me to display all the different filters the admin can use. The bottom part is divided between the actual list of students and a computer game the admin can play.

Creation of assets

All the assets in this project were created using DALL-E and Photoshop. This gave me the opportunity to establish a specific medieval aesthetic, and obtain elements as imagined by Hieronymous Bosch.

Hacking the system

The ultimate requirement for this project is to create a hacking feature. If you press the key "5", you will literally turn Hogwarts into flames. In addition, a new, mysterious student will be injected to the list with the added condition of being unexpellable.
