A personal project to improve my After Effects skills while playing with the A.I. image generator DALL-E.

Last December, Lionel Messi won the World Cup for Argentina. He's basically won everything there is to win. So what's next? This project aims to find an answer.

First iteration

The first iteration involved imagining how would different painters from every possible time period draw this incredible, historical event. From Monet to Da Vinci. From Bosch to Modigliani.

The evolution

The possibilities are endless. Entire museums could be filled up with these artworks. So what's next? Well.. what if these "paintings" came to life?
Introducing: Adobe After Effects.
